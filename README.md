**Ridiculously bad startup ideas, presented seriously.**

Shitups is a chaotic little platform where people anonymously post cursed startup ideas that absolutely should not get funding — but somehow still sound like a VC pitch deck.

## Features

- 🚀 Submit terrible startup ideas anonymously
- 🔥 Upvote the most unhinged ventures
- 💬 Anonymous comment section
- 😂 Emoji reactions 
- 🖼️ Optional image uploads
- 📱 Mobile responsive UI
- ⚡ Fast, simple, no framework


## Tech Stack

- **HTML**
- **CSS**
- **Vanilla JavaScript**
- **Supabase** (database + storage)

 


## Philosophy

Most startup platforms are designed to look professional.

Shitups does the opposite.

The goal is simple:
> Make the dumbest ideas sound revolutionary.

Examples:
- Uber for crying
- AI-powered breakup subscription
- NFT marriage certificates
- A cream that increases startup valuation
